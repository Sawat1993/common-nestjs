## Description

Starter Template for any __NodeJs/NestJS__ Application.
Integrated with __MongoDB__.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
